Civil Code

民法

総則

第1条（目的）
この法律は、私人間における財産関係及び責任関係を定め、自由で公正な社会秩序の形成を目的とする。
第2条（信義誠実の原則）
1. 権利の行使及び義務の履行は、信義に従い誠実に行わなければならない。 2. 権利は、これを濫用してはならない。

物権

第11条（物権の定義）
物権は、物を直接支配する権利とする。

債権

第19条（不法行為責任）
故意又は過失により、他人の権利又は法律上保護される利益を侵害した者は、損害を賠償する責任を負う。
Code of Civil Procedure

民事訴訟法

総則・管轄

第1条（目的）
この法律は、民事上の権利義務に関する紛争を、公正かつ迅速に解決するための手続を定めることを目的とする。
第2条（一審制の原則）
1. 民事裁判は、原則として一回の審理（第一審）によって終局的に解決される。 2. 本法に定める再審を除き、判決に対する不服申立ては認められない。

訴え・審理

第8条（訴状）
訴えは、次に掲げる事項を記載した訴状を裁判所に提出して提起する。 1. 当事者の表示 2. 請求の趣旨 3. 請求の原因

判決・執行

第15条（判決）
裁判所は、審理を終結したときは、判決を言い渡す。
第22条（執行）
確定判決は、強制執行の根拠となる。
